# Girls' Day 2026 – Deine Website mit KI

Slidev-Präsentation für den Girls' Day 2026 bei INNOQ (23. April 2026).

## Voraussetzungen

- [Node.js](https://nodejs.org/) (v18 oder neuer)
- [pnpm](https://pnpm.io/) (`npm install -g pnpm`)

## Installation

```bash
pnpm install
```

## Starten

```bash
pnpm dev
```

Die Präsentation ist anschließend unter [http://localhost:3030](http://localhost:3030) erreichbar.

## Weitere Befehle

| Befehl | Beschreibung |
|---|---|
| `pnpm dev` | Entwicklungsserver starten |
| `pnpm build` | Präsentation als statische Website bauen |
| `pnpm export` | Präsentation als PDF exportieren |
